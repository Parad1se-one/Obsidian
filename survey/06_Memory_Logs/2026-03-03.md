### 项目概述
**LLM-RL 记忆系统** - 基于 LangGraph + Qdrant + LLM 的强化学习记忆管理系统

**核心功能**:
- 智能记忆写入（基于重要性评分自动决策）
- 向量相似度检索（BGE-M3 模型，1024 维）
- 经验法则提炼（LLM 自动合并聚类记录）
- 时间衰减遗忘（定期清理低评分记录）

**技术栈**:
- 流程编排：LangGraph
- 向量数据库：Qdrant (:6333)
- 嵌入模型：BGE-M3 API (:8001)
- LLM: Qwen3 32B (:30620/v1)

**关键参数**:
```python
W1 = 0.6   # 奖励权重
W2 = 0.3   # 罕见度权重
W3 = 0.1   # 时间衰减权重
WRITE_THRESHOLD = 2.0
EVICTION_THRESHOLD = 0.5
CONSOLIDATE_THRESHOLD = 10
```

**评分公式**: `score = 0.6×|reward| + 0.3×rarity - 0.1×delta_t`
