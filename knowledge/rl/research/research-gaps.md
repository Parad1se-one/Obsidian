## 研究空白 1: 层次化世界模型

### 问题
- 当前模型预测固定步长 (通常 50-100 步)
- 无法进行长程规划
- 误差随时间累积

### 机会
```
层次化表示:
- 高层：抽象状态 (房间、目标)
- 中层：技能序列 (导航、操作)
- 低层：原始动作 (移动、抓取)
```

### 挑战
- 如何自动学习层次结构？
- 不同层次的时间尺度？
- 跨层次信息流？

---

## 研究空白 2: 预训练世界模型

### 问题
- 每个任务从头训练
- 无法利用已有知识
- 样本效率受限

### 机会
```
RL Foundation Model:
1. 大规模无标签视频预训练
2. 学习通用物理/动态
3. 下游任务快速适配
```

### 挑战
- 预训练目标设计？
- 如何评估预训练质量？
- 迁移机制？

---

## 研究空白 3: 因果世界模型

### 问题
- 当前模型学习相关性
- 无法进行因果推理
- 干预下泛化差

### 机会
```
因果世界模型:
- 学习结构因果模型 (SCM)
- 支持反事实推理
- 更好的域外泛化
```

### 挑战
- 如何从观测数据学习因果结构？
- 如何整合因果约束？
- 计算复杂度？

---

## 研究空白 4: 多 Agent 世界模型

### 问题
- 当前模型针对单 Agent
- 无法建模其他 Agent
- 多 Agent 场景性能差

### 机会
```
社会世界模型:
- 建模其他 Agent 的信念/意图
- 预测社会互动
- 支持合作/竞争
```

### 挑战
- 如何推断隐藏心理状态？
- 递归推理 (我想你认为...)？
- 可扩展性？

---

## 研究空白 5: 不确定性量化

### 问题
- 当前模型点估计
- 无法表达不确定性
- 高风险决策不可靠

### 机会
```
贝叶斯世界模型:
- 预测分布而非点
- 认知不确定性 + 偶然不确定性
- 支持风险敏感决策
```

### 挑战
- 计算效率？
- 高维潜空间贝叶斯推断？
- 如何训练？

## 推荐方向：预训练世界模型

### 理由
1. **影响力大:** 可能实现 RL 的"Foundation Model"突破
2. **时机成熟:** 大规模视频数据可用，计算资源充足
3. **与现有工作互补:** DreamerV3 等提供良好基础
4. **可行性高:** 可逐步推进，有明确里程碑

### 潜在贡献
- 首个大规模预训练世界模型
- 定义 RL 预训练基准
- 开源模型 + 代码
