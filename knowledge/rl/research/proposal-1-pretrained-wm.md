# 研究提案 1: Pretrained World Model (PWM)

**学习时间:** 2026-03-05 22:24-22:34
**主题:** 大规模预训练世界模型

---

## 标题

**PWM: Pretrained World Model for Sample-Efficient Reinforcement Learning**

---

## 核心思想

**假设:** 在大规模无标签视频数据上预训练世界模型，可以学习通用物理和动态，实现下游任务的快速适配。

```
预训练阶段:
- 数据：1000+ 小时视频 (机器人操作、游戏、物理模拟)
- 目标：预测下一帧 + 潜空间动态
- 输出：通用世界模型

适配阶段:
- 数据：少量任务特定交互 (<10K steps)
- 目标：微调 + 训练 Actor-Critic
- 输出：任务特定 policy
```

---

## 技术路线

### 1. 架构设计

```
基于 DreamerV3 扩展:

输入：视频帧 (可能 + 动作)
  ↓
ViT Encoder (预训练)
  ↓
RSSM (Recurrent State Space Model)
  ↓
预测：下一帧 + 奖励 + 终止
  ↓
ViT Decoder (可选)
```

**关键设计:**
- 使用 ViT 而非 CNN (更好的迁移性)
- 动作条件预测
- 多任务奖励头

---

### 2. 预训练目标

```python
loss = w1 * reconstruction_loss
     + w2 * dynamics_loss
     + w3 * contrastive_loss
     + w4 * action_prediction_loss
```

**创新点:**
- **对比学习:** 相似状态拉近，不同状态推远
- **动作预测:** 从状态变化推断动作 (无标签)
- **多尺度预测:** 预测 1 步、5 步、10 步后

---

### 3. 数据集

| 数据源 | 规模 | 类型 |
|--------|------|------|
| D4RG | 10M transitions | 机器人操作 |
| Atari 2600 | 57 games × 10M frames | 游戏 |
| DeepMind Control | 30 tasks × 1M frames | 控制 |
| YouTube (机器人) | 1000+ hours | 真实世界 |
| **总计** | **~1B frames** | 混合 |

---

### 4. 适配策略

```
方法 1: 微调
- 解冻全部参数
- 用任务数据继续训练
- 优点：性能好
- 缺点：可能遗忘

方法 2: 适配器
- 冻结主干
- 添加轻量适配器层
- 优点：保留预训练知识
- 缺点：容量受限

方法 3: 提示学习
- 学习任务嵌入 (prompt)
- 条件化世界模型
- 优点：高效
- 缺点：表达能力有限
```

## 实验设计

### 实验 1: Atari 100k 基准

**设置:**
- 预训练：Atari 57 游戏 (不含测试集)
- 适配：每个游戏 100K steps
- 基线：DreamerV3 (从头训练)

**指标:**
- 人类归一化分数
- 学习曲线 (样本效率)
- 零样本迁移能力

**预期:**
- 平均分数提升 20%+
- 样本效率提升 2x+

---

### 实验 2: 跨域迁移

**设置:**
- 预训练：模拟机器人 (DeepMind Control)
- 适配：真实机器人 (D4RG)
- 挑战：sim-to-real gap

**指标:**
- 成功率
- 样本效率
- 泛化到新物体/场景

**预期:**
- 相比从头训练，成功率提升 30%+
- 减少 5x 真实世界交互

---

### 实验 3: 少样本学习

**设置:**
- 预训练：混合数据
- 适配：新任务，仅 1K/10K/100K steps
- 基线：Meta-RL 方法

**指标:**
- 少样本性能
- 渐进学习曲线

**预期:**
- 1K steps: 达到基线 10K 水平
- 10K steps: 达到基线 100K 水平

## 创新点

### 1. 首个大规模 RL 预训练世界模型
- 类似 NLP 的 BERT/GPT
- 但针对序列决策

### 2. 多尺度预测目标
- 1 步、5 步、10 步联合预测
- 学习不同时间尺度动态

### 3. 动作推断预训练
- 从状态变化反推动作
- 无需动作标签

### 4. 系统基准与评估
- 定义 RL 预训练评估协议
- 开源代码与模型
