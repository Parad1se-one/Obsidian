## 🎯 候选蒸馏场景

### 场景 1: DQN 压缩 ⭐⭐⭐⭐⭐

```
教师：大 DQN (3 层 FC, 512 units)
学生：小 DQN (2 层 FC, 64 units)
任务：CartPole-v1
目标：压缩 10x, 性能保持 90%+
```

### 场景 2: SAC 策略蒸馏 ⭐⭐⭐⭐

```
教师：大 SAC (连续控制)
学生：小 SAC
任务：Pendulum-v1
目标：推理速度提升 5x
```

### 场景 3: 多任务蒸馏 ⭐⭐⭐⭐⭐

```
教师：多个任务特定策略
学生：单一通用策略
任务：Atari 子集 (Pong, Breakout, etc.)
目标：单一模型处理多任务
```
