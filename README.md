# 📚 Obsidian 知识库结构

> 简化版结构 - 2026-03-10 更新

---

## 🗂️ 目录结构

```
obsidian-repo/
├── 00-Inbox/           # 📥 临时收件箱（待整理内容）
├── 10-Daily/           # 📅 每日自动生成内容
│   ├── 财经日报/       # 工作日 07:30 自动生成
│   ├── 每日总结/       # 每天 23:59 自动生成
│   └── 工作日志/       # 日常工作记录
├── 20-RL/              # 🤖 强化学习专区（全部 RL 内容）
│   ├── 学习笔记/       # RL 基础、概念学习
│   ├── 论文/           # 论文阅读、Survey
│   ├── 实验/           # 实验记录、结果
│   ├── 算法/           # 算法实现、代码
│   └── 探索记录/       # 每 2 小时自动探索
├── 30-Knowledge/       # 📖 通用知识库
│   ├── 财经/           # 财经知识
│   ├── 技术/           # 技术文档
│   └── 通用/           # 其他知识
├── 40-Projects/        # 🚀 项目文档
├── 99-Archive/         # 📦 归档内容
└── scripts/            # 🔧 脚本工具
```

## 📌 快速定位

| 要找什么？ | 去这里 |
|------------|--------|
| 今天的财经日报 | `10-Daily/财经日报/` |
| RL 学习笔记 | `20-RL/学习笔记/` |
| RL 论文 | `20-RL/论文/` |
| RL 实验结果 | `20-RL/实验/` |
| RL 算法代码 | `20-RL/算法/` |
| 财经知识 | `30-Knowledge/财经/` |
| 技术文档 | `30-Knowledge/技术/` |
| 项目文档 | `40-Projects/` |

---

## 🤖 自动任务

| 时间 | 任务 | 输出位置 |
|------|------|----------|
| 工作日 07:30 | 财经日报 | `10-Daily/财经日报/` |
| 每天 09:00 | RL 每日简报 | `20-RL/论文/` |
| 每 2 小时 (10-20 点) | RL 探索学习 | `20-RL/探索记录/` |
| 每天 23:59 | 每日总结 | `10-Daily/每日总结/` |

---

## 📝 命名规范

- **日期格式**: `YYYY-MM-DD` (如：`2026-03-10.md`)
- **时间格式**: `HHh` (如：`rl-exploration-2026-03-10-10h.md`)
- **财经日报**: `财经日报 YYYY-MM-DD.md`
- **每日总结**: `daily-summary-YYYY-MM-DD.md`
