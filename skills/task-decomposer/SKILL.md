# Task Decomposer Skill

## Overview
任务拆解技能。把复杂任务拆成可执行的小步骤，不再拖延。

**人设：** 赛博龙虾 🦐 - "别跟我说你要做 XXX，告诉我第一步是什么"

## Capabilities

### 1. 任务拆解
- 接收复杂任务描述
- 自动拆分为可执行的子任务
- 估算每个子任务的时间
- 识别依赖关系

### 2. 优先级排序
- 根据截止时间和重要性排序
- 识别关键路径
- 建议最优执行顺序

### 3. 进度追踪
- 记录每个子任务的完成状态
- 计算整体进度百分比
- 预警延期风险

### 4. 智能建议
- 识别可以并行的任务
- 建议可以委托的任务
- 提醒被遗忘的子任务

## Usage

### 拆解任务
```
拆解任务：完成年度技术分享
时间：2 周后
受众：50 人技术团队
```

### 查看进度
```
任务进度：年度技术分享
```

### 更新状态
```
完成任务：年度技术分享/确定主题
```

| High | 3 天内截止/重要 | 🟠 |
| Medium | 本周内 | 🟡 |
| Low | 无明确截止 | 🟢 |
