## 调研时间
2026-03-05 17:15 - 小虾

---

## 🎯 核心需求
自动化的"财经日报"推送系统，包含：
1. **财经新闻采集** - 股票、市场、经济数据
2. **内容摘要生成** - AI 提炼重点
3. **格式化输出** - 结构化日报模板
4. **定时推送** - 每日自动发送

---

## 📊 现有工作流方案

### 方案 1: n8n + GPT-4 + NewsAPI ⭐ 推荐
**来源:** https://n8n.io/workflows/4375

**工作流:**
```
NewsAPI (每 10 分钟) → GPT-4 摘要 → Gmail 推送
```

**核心组件:**
| 组件 | 用途 | 成本 |
|------|------|------|
| NewsAPI | 获取头条新闻 | 免费 $0/月 (500 次/天) |
| GPT-4 | AI 摘要生成 | $0.01-0.03/次 |
| Gmail/n8n | 邮件推送 | 免费 |
| 定时触发 | 每 10 分钟/每日 | 免费 |

**优点:**
- ✅ 成熟工作流，有现成模板
- ✅ 可自定义新闻源和类别
- ✅ 支持个性化订阅列表

**缺点:**
- ❌ 需要部署 n8n 服务器
- ❌ 通用新闻，非财经专用

---

### 方案 2: Feedly + AI Feeds
**来源:** https://docs.feedly.com/article/692

**工作流:**
```
RSS 源 → Feedly Team Boards → AI 摘要 → 邮件模板 → 推送
```

**核心组件:**
| 组件 | 用途 | 成本 |
|------|------|------|
| Feedly | RSS 聚合 + AI 摘要 | $6-12/月 |
| Team Boards | 内容分类管理 | 包含在订阅中 |
| Automated Newsletters | 自动推送 | 包含在订阅中 |

**优点:**
- ✅ 专为新闻聚合设计
- ✅ 内置 AI 摘要功能
- ✅ 支持品牌定制模板

**缺点:**
- ❌ 付费服务
- ❌ 财经源需要手动配置

---

### 方案 3: Finlight API + 自定义工作流 ⭐ 财经专用
**来源:** https://finlight.me/

**工作流:**
```
Finlight API (财经新闻) → 情感分析 → 格式化 → 推送
```

**核心组件:**
| 组件 | 用途 | 成本 |
|------|------|------|
| Finlight API | 财经新闻 + 情感分析 | 联系定价 |
| WebSocket | 实时新闻流 | 包含 |
| SDK | Python/JS 集成 | 免费 |

**优点:**
- ✅ **专为财经设计** (股票、加密货币、外汇)
- ✅ 内置情感分析 (positive/negative)
- ✅ 实时新闻流
- ✅ 公司实体识别 (ticker:NVDA, ticker:SPY)

**缺点:**
- ❌ 需要开发集成
- ❌ 定价不透明

### 方案 4: FinancialNewsAPI + Python
**来源:** https://github.com/FinancialNewsAPI/financial-news-api-python

**工作流:**
```
FinancialNewsAPI (5000 万 + 文章) → Python 脚本 → 摘要 → 推送
```

**核心组件:**
| 组件 | 用途 | 成本 |
|------|------|------|
| FinancialNewsAPI | 财经新闻数据库 | 联系定价 |
| Python SDK | 数据获取和处理 | 免费 |
| 自定义脚本 | 摘要和推送逻辑 | 自研 |

**优点:**
- ✅ 5000 万 + 历史文章
- ✅ 实时新闻流 API
- ✅ Python 集成简单 (<10 行代码)

**缺点:**
- ❌ 需要自研摘要和推送逻辑

## 🔧 技术实现

### API 配置需求

| API | 用途 | Key 获取 | 免费额度 |
|-----|------|---------|---------|
| **Finlight API** | 财经新闻 + 情感分析 | https://finlight.me | 需联系 |
| **NewsAPI** | 通用新闻补充 | https://newsapi.org | 500 次/天 |
| **Alpha Vantage** | 股票行情数据 | https://alphavantage.co | 5 次/分钟 |
| **Twelve Data** | 技术指标数据 | https://twelvedata.com | 800 次/天 |

### 脚本结构

```bash
skills/financial-daily/
├── SKILL.md                 # 技能文档
├── financial-daily.sh       # 主脚本
├── collectors/
│   ├── finlight-collector.sh
│   ├── newsapi-collector.sh
│   └── rss-collector.sh
├── processors/
│   ├── summary-generator.sh
│   └── sentiment-analyzer.sh
├── templates/
│   └── daily-report.md      # 日报模板
└── config/
    └── sources.json         # 新闻源配置
```

### Cron 配置

```bash
# 每个工作日早上 8:00 生成财经日报
0 8 * * 1-5 openclaw send "生成财经日报"
```
